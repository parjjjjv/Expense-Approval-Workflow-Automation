# Expense Approval Workflow Automation

## 🚀 Overview
Automates faculty expense approvals using Google Apps Script.

## 🔧 Features
- Approval/Rejection system
- PDF generation
- Email notifications
- Budget validation
- Calendar automation

## 🧠 Architecture
- Google Forms → Input
- Google Sheets → Database
- Apps Script → Logic
- Google Docs → PDF Template
- Gmail → Notifications
- Calendar → Monthly events

## 📁 Project Structure
- scripts/ → Core logic
- docs/ → Report & screenshots
- assets/ → Demo
